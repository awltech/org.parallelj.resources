org.parallelj.resources by XA-Labs

This repository allows to store various resources related to ParallelJ development. It will contain different kind of resources such as documentation, development tools, and so on.

Authors

Antoine Neveux

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/XA-Labs/org.parallelj.resources

Finally, you can install the ParallelJ Resources in Eclipse using this update-site:
http://xa-labs.github.com/org.parallelj.resources/p2/

get the source code on GitHub : XA-Labs/org.parallelj.resources
