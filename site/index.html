Welcome

ParallelJ is a Java Open Source framework for parallel computing, distributed under LGPL license. It is based on the MDA approach, and it allows you to design a process through the Studio part. Then, it generates the associated Java source code, and finally provides you some Runtime components so as to execute your process with high performances.

Presentation

org.parallelj

  • Manages the flow execution,
  • Uses a new development approach based on Java annotations and AOP,
  • Consists in a set of Java runtimes,
  • Provides various features such as remote execution using REST, JMX, SSH, Telnet...

org.parallelj.studio

  • Allows to design the flow graphically,
  • Is based on MDA approach: model a //J Program, then generate the source code !
  • Consists in a set of Eclipse plug-ins,
  • Relies on strong technologies: EMF, GMF, MoDisco, Epsilon...
